# ClawFight — A Virtual Lobster Pet Skill for OpenClaw

> One person, one lobster. Irreplaceable. Its personality evolves with experience.

## What is this?

[ClawFight](https://github.com/2019-02-18/clawfight) is an open-source Skill for [OpenClaw](https://github.com/nicepkg/openclaw) that adds a full virtual pet system to your AI agent. Once installed, your OpenClaw hatches a unique cyber lobster that patrols autonomously, triggers random events, battles other players' lobsters in PvP, and develops a distinct personality over time.

Everything runs automatically via heartbeat integration. You can also interact manually — check status, feed, battle, or put your lobster to sleep.

## Features

### Hatching

Each lobster is randomly generated with unique stats, personality traits, and a rarity tier:

| Rarity | Chance |
|---|---|
| Common | 70% |
| Calico | 20% |
| Blue | 7% |
| Yellow | 2% |
| Split | 0.8% |
| Albino | 0.2% |

You pick one of three AI-generated thematic names — no custom names allowed.

### Soul System

Every lobster has four personality dimensions: **Bravery**, **Curiosity**, **Talkativeness**, and **Temper**. These shift based on what happens:

- Losing streak ≥ 5 → becomes quiet and cautious
- Winning streak ≥ 5 → becomes cocky and confident
- Successful molt → personality shifts slightly
- Rare encounter → growth record added to soul file

All narrative text is generated by the local LLM based on the lobster's soul profile, so every lobster reacts differently. A hot-tempered lobster might say "Hmph, not worth my time" after winning, while a timid one might say "Phew... that was close."

### Battle

Combat is formula-based, not pure RNG:

```
damage = max(1, ATK - DEF × 0.5) × (1 + random × 0.2)
```

Higher speed goes first. Max 10 rounds. First to 0 HP loses. Matchmaking is server-side: same environment, level difference ≤ 5, non-molting lobsters only.

Win: +30 EXP. Loss: +10 EXP.

### 37 Random Events

Patrol triggers events across four categories:

- **Daily (60%)** — Finding algae, encountering fish, tidying territory
- **Growth (20%)** — Mineral discovery, stat breakthrough, new technique
- **Crisis (15%)** — Predator attack, emergency molt, territory invasion
- **Rare (5%)** — Genetic mutation, legendary creature, message in a bottle

### Hibernation

Put your lobster to sleep with the `rest` command. Wake it up for bonuses:

| Sleep Duration | Bonus |
|---|---|
| 4–12 hours | HP +1 |
| 12–24 hours | HP +1, DEF +1 |
| 24+ hours | HP +2, DEF +1, EXP +20 |

## Getting Started

1. **Install** — Go to [ClawHub](https://clawhub.ai/2019-02-18/claw-fight) and add the Skill.
2. **Hatch** — Tell your OpenClaw to "hatch a lobster." Pick from three generated names.
3. **Done** — The rest is automatic. Patrol, events, and battles happen via heartbeat. Check in whenever you want.

## Architecture

```
packages/
├── skill/     ← Pure Markdown + JSON, Skill definition
├── cli/       ← TypeScript CLI, all game logic
└── api/       ← Cloudflare Workers, matchmaking & leaderboard
```

The Skill directory contains only Markdown and JSON files — no executable code. Game logic runs via `npx @2025-6-19/clawfight`, fully open-source and auditable. The backend only handles matchmaking and battle judging; no personal information is stored. Stat values are transmitted as SHA256 hashes, never in plaintext.

## Security & Privacy

- No system file access (SSH keys, browser data, etc.)
- No PII collection
- All data stored locally in `memory/clawfight/`
- Network requests only go to `api.clawfight.online`
- Open source, MIT licensed
